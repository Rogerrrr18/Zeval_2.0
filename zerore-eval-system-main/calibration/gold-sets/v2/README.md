# Gold Set V2

This directory is generated by `npm run gold:v2:scaffold`.

## Files

- `cases.jsonl`: case records used by judge calibration.
- `annotation-tasks.jsonl`: assignable task index for labelers and reviewers.
- `label-drafts/*.json`: one editable draft per task.
- `labels.jsonl`: generated only after approved drafts are imported.
- `import-report.md`: generated import gate report.

## Current Scope

- Cases: 4
- Assigned Tasks: 4
- Tasks With Reviewer: 4

## Workflow

1. Assign owners in `annotation-tasks.jsonl` or regenerate with `--assignees` / `--reviewers`.
2. Labelers fill their draft files under `label-drafts/`.
3. Reviewers set `reviewStatus` to `approved` only after evidence and scores are checked.
4. Run `npm run gold:v2:import` to produce `labels.jsonl`.
5. Run `npm run calibration:judge -- --cases calibration/gold-sets/v2/cases.jsonl` and then agreement against the imported labels.
